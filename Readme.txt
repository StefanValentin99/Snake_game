Joc Snake facut de mine dupa modelul prezentat de Wajiha Urooj in articolul de pe blogul ei: https://www.edureka.co/blog/snake-game-with-pygame/
Acest proiect a fost facut cu scopul de a invata si de a experimenta modul in care se pot face jocuri in Python folosind module precum "pygame", "time" sau "random"
O scurta descriere a modului in care functioneaza jocul:
De fiecare data cand sarpele "mananca" un fruct (generat random pe ecran), corpul lui se mareste cu o unitate si un alt fruct este generat pe ecran
Daca sarpele intra in propriul corp - capul primeste aceleasi coordonate cu cele ale unui title din corpul sau - sau iese din ecran, jocul se va sfarsi
